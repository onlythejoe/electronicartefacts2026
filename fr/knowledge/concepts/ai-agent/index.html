CONCEPT

Agent d’IA

An AI agent est un software système dans which a modèle can select actions, use outils, observe results et adapt its next steps toward an objective within defined boundaries.

AI agents combine modèles, outils, mémoire, environmental feedback, stopping conditions, permissions et evaluation dans an iterative action loop.

Editorial claims

Position maintained by Electronic Artefacts

  1. Agency est un architectural property de the whole système, not a property de the modèle output alone.
  2. Agents require stronger permissions, observability et evaluation than deterministic workflows.

Boundaries

What this concept excludes

  1. A fixed sequence de modèle calls avec no modèle-directed action selection
  2. Unbounded autonomy avec no observable state ou control surface
