CONCEPT

Rétroaction cybernétique

Cybernetic feedback est le circulation de information about a système's state back into the système so that behavior can be regulated, adapted ou transformed.

Cybernetic feedback relie control, communication, systèmes theory, generative art, machine listening, adaptive interfaces et runtime gouvernance.
