CONCEPT

Graphisme procédural

Procedural grapheics sont images ou visuel systèmes generated from rules, algorithms, parameters, simulation ou code rather than only manually drawn assets.

Procedural grapheics connect shaders, generative systèmes, motion design, programmation créative, visuel culture et real-time web interfaces.

Editorial claims

Position maintained by Electronic Artefacts

  1. Procedural grapheics sont strongest when parameters express a visuel grammar rather than arbitrary variation.
  2. Preservation de procedural visuels requires code, parameters, runtime context et output records.

Boundaries

What this concept excludes

  1. Static illustration exported from manual design software avec no procedural layer
  2. Randomized decoration avec no coherent visuel système
