CONCEPT

Audio web

Web audio est le use de navigateur-based audio APIs et related web technologies à synthesize, process, analyze, route et present sound.

Web audio relie sound design, navigateur runtimes, interactive médias, programmation créative et public sound outils.

active validated v1.0.0

Editorial claims

Position maintained by Electronic Artefacts

  1. Web audio turns the navigateur into a viable medium pour interactive sound systèmes.
  2. Navigateur sound work needs careful attention à latency, permissions, performance et preservation.

Boundaries

What this concept excludes

  1. Embedded audio playback avec no processing ou interaction layer
  2. Native audio software avec no navigateur runtime
