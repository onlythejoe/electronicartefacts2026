## Editorial rationale

The selected topics are evergreen because they describe standards, protocols and implementation patterns rather than product releases. They also create practical routes for future Electronic Artefacts projects:

- VASTE can expose graph-scoped resources and tools through MCP.
- VASTE can observe runtime behavior with OpenTelemetry-style traces and events.
- Palimpsests can use C2PA as media provenance infrastructure.
- Vestiges can use IIIF for compound cultural object presentation.
- Browser-local creative tools can use WebNN when capability support is available.
