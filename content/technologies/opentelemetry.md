## Role

OpenTelemetry supports operational understanding by standardizing how systems describe requests, spans, logs, metrics, errors and service context.

## Use

Electronic Artefacts can use OpenTelemetry as a reference for observing VASTE event propagation, AI tool calls, graph projections and user-visible runtime behavior.
