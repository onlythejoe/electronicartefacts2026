## Role

The Web Audio API provides graph-based audio routing and processing in browsers. It is relevant to interactive essays, sound analysis, synthesis and public listening tools.

## Use

Electronic Artefacts can use it for ORETH prototypes, audio visualizers, browser-native sound experiments and educational demonstrations.
