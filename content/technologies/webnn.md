## Role

WebNN defines a web API for running neural-network inference through available platform acceleration while keeping application code inside the browser environment.

## Use

Electronic Artefacts can use WebNN as a reference for browser-local analysis, interactive creative tools, private archive assistants and accessible multimodal interfaces.
