## Description

The repository contains the open-source implementation of Voice Capture Studio: a static browser application for guided speech recording, local workspace state and structured WAV plus JSON exports.

## Provenance

Electronic Artefacts publishes and maintains the repository at [github.com/electronicartefacts/voice-capture-studio](https://github.com/electronicartefacts/voice-capture-studio). The live application is deployed through GitHub Pages at [electronicartefacts.github.io/voice-capture-studio](https://electronicartefacts.github.io/voice-capture-studio/).

## Significance

This code artefact makes the product inspectable. It preserves the architecture doctrine, system model, corpus structure, workspace structure, export structure, tests, contribution guidelines and deployment workflow used to evaluate the application.

## Rights

Voice Capture Studio is released under the MIT License. User voice data and generated workspaces do not belong in the repository.
