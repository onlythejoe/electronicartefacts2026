## Définition

L’enregistrement vocal est la capture contrôlée de parole pour une révision, une édition, une conservation ou une préparation de dataset ultérieure.

## Périmètre

Le concept inclut les prompts, les informations de locuteur, le contexte micro, le silence de pièce, les prises, les transcriptions, le timing, les contrôles qualité et les métadonnées d’export.

## Applications

Electronic Artefacts l’utilise pour décrire les outils de capture vocale dans le navigateur, les recherches audio proches d’ORETH et les archives vocales structurées.

## Limites

L’enregistrement vocal n’entraîne pas lui-même un modèle. Il produit un matériau source dont l’usage ultérieur dépend de décisions séparées de consentement, de qualité et de traitement.
