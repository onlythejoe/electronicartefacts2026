## Problème

Une lecture de l’open source comme infrastructure culturelle, à travers les licences, la maintenance, les outils créatifs, les chaînes de dépendances, les communs et les systèmes publics de connaissance.

## Architecture

L’open source est une infrastructure culturelle parce que le code partagé détermine ce que les artistes, chercheurs et développeurs peuvent construire. L’accès au code ne suffit pas : licences, maintenance, gouvernance et documentation sont tout aussi décisifs.
