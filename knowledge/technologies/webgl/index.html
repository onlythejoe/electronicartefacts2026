WebGL

WebGL is a web platform technology for rendering interactive 2D and 3D graphics in compatible browsers using GPU acceleration.

WebGL makes GPU-accelerated graphics available on the web. It is relevant to shaders, procedural visual systems and interactive public artefacts.
