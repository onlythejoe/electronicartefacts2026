JSON-LD

JSON for Linked Data

JSON-LD is a W3C format for serializing Linked Data in JSON, making semantic graph metadata compatible with common web programming workflows.

Role

JSON-LD serializes Linked Data as JSON. It is practical for static websites because build tools can generate it alongside HTML without requiring a graph database at runtime.

Use

Electronic Artefacts uses JSON-LD to expose each public entity as structured data. Publications become articles, concepts become defined terms, programs become software applications or creative works, and identifier routes expose machine-readable records.

Limitations

JSON-LD improves machine interpretation, but it cannot repair weak content. The structured data should match visible human content and remain consistent with canonical routes.
