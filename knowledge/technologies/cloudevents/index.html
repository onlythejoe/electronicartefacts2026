TECHNOLOGY

CloudEvents

CloudEvents is a CNCF specification for describing event data in a common, portable envelope.

active canonical v1.0.0

Role

CloudEvents standardizes common event metadata so that producers, transports and consumers can exchange event records with fewer platform-specific assumptions.

Use

Electronic Artefacts can use it as a reference when defining VASTE event envelopes and public event projections.

References

See the CloudEvents specification and Event-Driven Architecture.
