C2PA

C2PA is a technical standard family for attaching verifiable provenance and content-credential metadata to media assets.

Technology role

Why this technology matters here

C2PA provides signed media provenance manifests that can record origin, actions, assertions and trust information for images, audio, video and documents.

Role

C2PA gives media producers and tools a way to bind provenance assertions to assets so later systems can inspect source and transformation history.

Use

Electronic Artefacts can treat C2PA as a reference technology for archive trust, generative-media labelling, Palimpsests documentation and production-memory workflows.

References

See the C2PA specification set, including Content Credentials, implementer guidance, UX guidance, security considerations and AI/ML guidance.
