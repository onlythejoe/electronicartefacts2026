IIIF

IIIF is a framework of APIs for presenting, structuring and annotating digital representations of cultural and archival objects.

Role

IIIF standardizes the presentation of digitized and born-digital cultural objects so viewers, archives and institutions can exchange compound object structures.

Use

Electronic Artefacts can use IIIF concepts to reason about Vestiges objects, Palimpsests dossiers, audio-visual collections and annotation-aware public records.

References

See the IIIF Presentation API 3.0 specification for collections, manifests, canvases, ranges and annotations.
